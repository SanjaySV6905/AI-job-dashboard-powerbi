AI Job Market Analysis: Power BI Dashboard
An interactive and comprehensive data analysis project exploring global trends in AI job salaries, required skills, and location preferences, visualized through a professional Power BI Dashboard.

This project transforms raw job posting data into actionable insights for the Artificial Intelligence sector.

The dashboard analyzes:

Global Salary Distribution by company location (e.g., Switzerland, UK, Germany).
Average Salary  based on Experience Level (Entry, Mid, Senior, Executive).
Remote Work  popularity and ratio (On-Site vs. Hybrid vs. Fully Remote).
Top Paying Skills  required for various AI-related job titles.
